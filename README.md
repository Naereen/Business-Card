# Business Card
> Forked from [@jilljenn/business-card](https://github.com/jilljenn/business-card)

![Some business card in LaTeX](card.png)

## Requirements
- XeLaTeX
- Logo of your institutions

## Build
```bash
xelatex card
```
Or by using [the Makefile](Makefile):
```bash
make
```
